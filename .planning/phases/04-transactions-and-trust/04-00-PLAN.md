- listing_id: BIGINT NOT NULL (FK to listings)

- seller_id: BIGINT NOT NULL (FK to users)

- amount: DECIMAL(10,2) NOT NULL

- cancellation_reason: VARCHAR(500)
- version: BIGINT (optimistic locking per D-44)

**ledger_entries table:**
- id: BIGINT PRIMARY KEY AUTO_INCREMENT
- transaction_id: BIGINT NOT NULL (FK to transactions)
- account_type: VARCHAR(30) NOT NULL (BUYER_ESCROW, SELLER_PENDING, PLATFORM_FEE)
- entry_type: VARCHAR(10) NOT NULL (DEBIT, CREDIT)
- amount: DECIMAL(10,2) NOT NULL
- idempotency_key: VARCHAR(64) UNIQUE
- created_at: TIMESTAMP NOT NULL

**ratings table:**
- id: BIGINT PRIMARY KEY AUTO_INCREMENT
- transaction_id: BIGINT NOT NULL (FK to transactions)
- rater_id: BIGINT NOT NULL (FK to users)
- rated_user_id: BIGINT NOT NULL (FK to users)
- rating: INT NOT NULL (1-5 per D-35)
- review_text: VARCHAR(500) (optional per D-36)
- is_visible: BOOLEAN NOT NULL DEFAULT FALSE (blind rating per D-30)
- created_at: TIMESTAMP NOT NULL

**disputes table:**
- id: BIGINT PRIMARY KEY AUTO_INCREMENT
- transaction_id: BIGINT NOT NULL (FK to transactions)
- opener_id: BIGINT NOT NULL (FK to users)
- reason: VARCHAR(100) NOT NULL
- description: TEXT NOT NULL
- status: VARCHAR(30) NOT NULL (OPEN, UNDER_REVIEW, WAITING_BUYER_EVIDENCE, WAITING_SELLER_EVIDENCE, RESOLVED_BUYER, RESOLVED_SELLER, PARTIALLY_RESOLVED, ESCALATED, CLOSED per D-18)
- resolution: TEXT
- created_at: TIMESTAMP NOT NULL
- updated_at: TIMESTAMP
- resolved_at: TIMESTAMP

**Add columns to users table for rating aggregation (per D-41):**
- average_rating: DECIMAL(3,1)
- total_ratings: INT DEFAULT 0

**Indexes:**
- idx_transactions_listing on transactions(listing_id)
- idx_transactions_buyer on transactions(buyer_id)
- idx_transactions_seller on transactions(seller_id)
- idx_transactions_status on transactions(status)
- idx_transactions_idempotency on transactions(idempotency_key)
- idx_ledger_transaction on ledger_entries(transaction_id)
- idx_ratings_transaction on ratings(transaction_id)
- idx_ratings_rated_user on ratings(rated_user_id)
- idx_disputes_transaction on disputes(transaction_id)

**Foreign Keys:**
- fk_transactions_listing to listings(id)

- fk_transactions_conversation to conversations(id)
