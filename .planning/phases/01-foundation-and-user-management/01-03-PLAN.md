  **AvatarService.java (package: com.tradingplatform.avatar):**
  - @Service, @RequiredArgsConstructor
  - Inject AvatarStorageService
  - @Value("${avatar.upload.dir:./uploads/avatars}") String uploadDir
  - @Value("${avatar.max-size:5242880}") long maxFileSize (5 MB default, D-10)
  - Static final Set<String> ALLOWED_TYPES = Set.of("image/jpeg", "image/png", "image/webp") (D-11)

  - Method: storeAvatar(MultipartFile file, Long userId) returns String (filename)
    - Call validateFile(file)
    - Generate filename: "user_{userId}.{extension}"
    - Read image, create thumbnail (200x200, center-cropped)
    - Delete old avatar if exists (D-12 - replace, not gallery)
    - Save thumbnail to uploadDir
    - Return filename

  - Method: validateFile(MultipartFile file) throws ApiException
    - Check not empty
    - Check size <= maxFileSize
    - Check content type in ALLOWED_TYPES
    - Throw ApiException with INVALID_AVATAR error code and descriptive message

  - Method: createThumbnail(BufferedImage original) returns BufferedImage
    - Center crop to square
    - Resize to 200x200
    - Use bilinear interpolation for quality

  - Method: deleteAvatar(String filename)
    - Delete file from uploadDir if exists

  **AvatarStorageService.java (package: com.tradingplatform.avatar):**
  - @Service
  - @Value("${avatar.upload.dir:./uploads/avatars}") String uploadDir

  - Method: init() - create uploadDir if not exists (call on @PostConstruct)
  - Method: store(String filename, BufferedImage image, String format) - write to disk
  - Method: load(String filename) returns Path - resolve path in uploadDir
  - Method: delete(String filename) - delete file
  - Method: exists(String filename) returns boolean
